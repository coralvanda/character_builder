What is this?
-------------
This is an independent project created to experiment with and learn about various concepts in programming using Python, and to learn to use the tkinter module for creating GUIs.

How to use
----------
Save the RPGcharBuilder.py and race.py files together in the same directory, and use Python3 to run the RPGcharBuilder.py file.  

The program allows the user to save a configuration based on the name provided, and it will automatically create a new sub-directory to store the save files.
